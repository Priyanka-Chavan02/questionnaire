Developed by Brendan Eich in 1995, JavaScript is one of the most popular languages for web development.
        It was initially created to develop dynamic web pages. Every JS program is called a script, which can be attached to any web page’s HTML. 
        These scripts run automatically when the page loads.
        A language which was initially used to create dynamic web pages, can now be executed on the server and practically on any device consisting of the 
        JavaScript Engine.
